# Metodi di risoluzione delle equazioni di ricorrenza
- Iterativo/albero di ricorsione (sia ricorsivi che divide et impera)
- "Dell'esperto" (solo divide et impera)

# Esercizi
Scrivere un algoritmo che conti le coppie caratteri consecutivi in un'array di lettere.
  
```swift
Int conta(Int a[], Int i, Int f)
{
    If (i == f) Return 0;
    If (i == f - 1)
    {
        If (a[i] == a[f])
        {
            Return 1;
        }
        Else
        {
            Return 0;
        }
    }

    Int m = (i + f) / 2;
    Int r1 = conta(a, i, m);
    Int r2 = conta(a, m + 1, f);
    Int r3;

    If (a[m] == a[m + 1])
    {
        r3 = 1;
    }
    Else
    {
        r3 = 0;
    }

    Return r1 + r2 + r3;
}
```

**Equazione di ricorrenza:**

Casi base:
- ($n = 1$): 2 istruzioni
- ($n = 2$): 4 istruzioni

Caso generale: $3 + T(n / 2) + T(n / 2) + 3 = 2T(n/2) + 6$

**Tempo di esecuzione:**

Usiamo metodo dell'esperto:

- $f(n) = 6 = \Theta(1)$
- $n^{log_b a} = n^1 = \Theta(n)$

$f(n)$ è $O(n^{log_b a - \epsilon})$? ✅<br>
Siamo nel caso 1 del metodo dell'esperto:
$T(n) = \Theta(n^{log_b a}) = \Theta(n)$
