# Esempi di algoritmi ricorsivi
Algoritmo ricorsivo per calcolare $a^n$ (con $a > 1, n > 1$)
```swift
Int Potenza(Int a, Int n)
{
	If (n == 0)
	{
		Return 1;
	}
	Else
	{
		Int r = Potenza(a, n - 1);
		Int tot = a * r;
		Return tot;
	}
}
```

**Analisi del tempo d’esecuzione**
Sia $T(n)$ il tempo richiesto da `Potenza()`.

- `If (n == 0)`: $1$
- `Return 1`: $t_{if}$
- `Int tot = a * r`: $f_{if}$
- `Return tot`$f_{if}$

Ma `Int r = Potenza(a, n - 1)`?
- Non ha tempo costante, perché è un’altra funzione
- Ogni volta chiamo ancora `Potenza()`

Quindi ho $T(n-1) \cdot f_{if}$:

$T(n) = 3C + T(n - 1)$ se $n \geq 0$

$T(n) = 2C$ se $n = 0$

Dato un’array di lettere, contare quante volte compare la lettera Z.
```swift
Int CountZ(Char v[], Int i)
{
	If (i > Length(v))
	{
		Return 0;
	}
	Else
	{
		Int b = 0;
		If (v[i] == 'Z')
		{
			b = 1;
		}
		
		Int r = CountZ(v, i + 1);
		Return b + r;
	}
}
```

**Analisi del tempo d’esecuzione**
- `If (i > Length(v))`: $1$
- `Return 0`: $t_{if}$
- `Int b = 0`: $f_{if}$
- `If (v[i] == ‘Z’)`: $f_{if}$
- `b = 1`: $t_{if2}$
- `Int r = CountZ(v, i + 1)`: $T(n - 1)$
- `Return b + r`: $f_{if}$

Caso base: $T(n) = 2C$ se `i > Length(v)`

Caso generale: $T(n) = 4C + T(n - 1)$

Dato un array di caratteri, dire se le lettere costituiscono una parola palindroma
```swift
Bool Palindrome(Char v[], Int start, Int end)
{
	If (start == end Or start > end) 
	{
		Return True;
	}
	Else
	{
		If (v[start] != v[end])
		{
			Return False;
		}
		Else
		{
			Boolean r = Palindrome(v, start + 1, end - 1);
			Return r;
		}
	}
}
```

**Analisi dei tempi d’esecuzione**
- Caso base: $T(n) = 2C$ se $n \leq 1$
- Caso migliore: $T(n) = 3C$ se `v[1] != v[n]`
- Caso peggiore: $T(n) = 3C + T(n - 2)$

Dato un array di numeri interi, dire quante volte un elemento è seguito dal suo successore
```swift
Int Successore(Int v[], Int i)
{
	If (p >= Length(v))
	{
		Return 0;
	}
	Else
	{
		Int b = 0;
		If (v[p] + 1 == v[p + 1])
		{
			b = 1;
		}
		
		Int r = Successore(v, p + 1);
		Return b + r;
	}
}
```
